In this exercise, you will use the skills you've learned in the course to make a R Markdown report that provides answers to a set of coding tasks.

You will upload a .html version of your R Markdown report as your submission.

Review criteria

A classmate will download your report and compare the output generated by your file against the output in the solution key.

Step-By-Step Assignment Instructions

Download the .rmd file attached to these instructions.

Review the solution key here: https://rpubs.com/cpaschall/c3p1

Review the .rmd file. It contains starter code for the assignment.

Complete the coding tasks in the spaces indicated in the .rmd file. Try to make your output match the output from the solution key.

When you are done, compile your report using the knit button as a .html file. Save it on your computer. This is what you will submit.
