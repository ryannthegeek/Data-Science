The following lessons use gganimate. After the course was created, there was an update to gganimate, which is causing an error with the visualizations generated by the sample code in the video. You may see a green flashing effect using the lecture code.

A fix for this in the works, but in the meantime, I recommend this approach to your animations:

#Install the magick package

install.packages("magick")

#Load the required libraries:

library(gganimate)

library(magick)

fig <-ggplot(....[your animation code, using gganimate]

animate(fig,renderer=magick_renderer())
